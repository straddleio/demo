# Paykey Generator Server Requirements
#
# This is a minimal Python 3.x HTTP server with no external package dependencies.
# All cryptographic operations use the b3sum binary (BLAKE3 tool).
#
# System Requirements:
# - Python 3.7+
# - b3sum (BLAKE3 command-line tool)
#
# Installation on Arch Linux:
#   sudo pacman -S b3sum
#
# No pip packages required - uses only Python stdlib:
# - http.server (HTTP server)
# - json (JSON parsing/serialization)
# - subprocess (b3sum binary invocation)
# - urllib.parse (URL parsing)
# - datetime (timestamp generation)
# - tempfile, os (temporary file handling for keyed mode)
